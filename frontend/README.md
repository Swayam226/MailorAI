# MailorAI - Frontend

AI-powered cold outreach generation platform built using the MERN, Groq AI, JWT authentication, and OTP email verification.

## Live

Full App: https://mailor-ai.vercel.app/

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- Axios
- React Router

## Local Setup

### Clone Repo

```bash
git clone <repo-url>
```

### Frontend setup

```bash
cd frontend
npm install
npm run dev
```

### Frontend

```env
VITE_API_BASE_URL=
```
